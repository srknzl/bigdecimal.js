# Security Policy

## Supported Versions

Only the latest published version of `bigdecimal.js` on npm receives security fixes.

| Version | Supported          |
| ------- | ------------------ |
| 1.5.x   | :white_check_mark: |
| < 1.5   | :x:                |

## Reporting a Vulnerability

Please report security vulnerabilities privately using [GitHub Security Advisories](https://github.com/srknzl/bigdecimal.js/security/advisories/new) rather than opening a public issue.

If that isn't possible, email serkanozel380@gmail.com with a description of the issue, steps to reproduce, and its potential impact.

You should receive a response within a few days. If the report is confirmed, a fix will be released as a patch version on npm and credited in the release notes, unless you prefer to remain anonymous. If declined, you'll get an explanation why.
